# student-score-prediction-ml
A Machine Learning model built using Linear Regression to predict student exam scores based on study hours.
🎯 Student Score Prediction using Machine Learning
📌 Project Overview

This project predicts student marks based on study hours using Machine Learning.

The goal is to help an EdTech platform identify students who may need support before exams.

📊 Dataset
The dataset contains:
Study Hours
Student Scores

🤖 Model Used
Linear Regression

⚙️ Technologies Used
Python
Pandas
NumPy
Matplotlib
Scikit-learn

📈 Example Prediction
Input:
Study Hours = 4
Output:
Predicted Score ≈ 75

🚀 How to Run the Project
Clone the repository
Install required libraries:
pip install -r requirements.txt
Run the notebook
